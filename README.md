# 📖 About

This project analyzes IPL historical data to uncover trends related to:

- Winning teams
- Toss decisions
- Player of the Match
- Venues
- Match outcomes
- Team performance
- Season statistics

The notebook performs complete EDA from raw CSV to meaningful visualizations.

---

# ✨ Features

- Data Loading
- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis
- Statistical Summaries
- Matplotlib Visualizations

- Automated CSV Loader (`load_data.py`)
